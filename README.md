# wx-news: 极简的微信新闻小程序

## 概述

微信小程序入门项目，使用了Udacity的虚拟[新闻API](https://github.com/udacity/wmpnd-news/blob/master/news_api.md)

## 已实现：
- [x] 新闻列表、分类图文显示、详情页等基础功能。
- [x] 使用 wx:for 动态的显示新闻分类部分，且导航栏浮动。
- [x] 特定页面下拉手势激活刷新功能。
